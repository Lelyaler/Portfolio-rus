Обо мне

Я веб-разработчик с двухлетним опытом создания сайтов. В настоящее время работаю фронтенд-разработчиком в компании MaxBitSolution (iGaming), где занимаюсь разработкой адаптивных веб-интерфейсов, а также созданием и кастомизацией шаблонов на WordPress. Параллельно с работой в найме я готова брать проекты на фрилансе, помогая клиентам реализовывать идеи.

Я изучила необходимые технологии самостоятельно и применила их на практике, не прибегая к рекламным IT-курсам. Развиваюсь в области фронтенд-разработки.

Обладаю хорошими навыками в HTML, CSS и JavaScript. Занимаюсь разработкой сайтов "под ключ", включая: приобретение хостинга и домена, интеграцию верстки на CMS платформу WordPress, работу с хуками PHP и настройку админ-панели. Клиент сможет обновлять актуальную информацию на сайте без знаний кода. Я готова создать сайт по вашим референсам: даже если у вас нет готового макета, достаточно иметь представление о том, как должен выглядеть и работать ваш будущий сайт. Мы вместе обсудим идеи, и я предложу оптимальное решение, которое будет соответствовать вашим ожиданиям.

Мои ключевые навыки:

HTML и CSS: Уверенное знание семантического HTML, методологии БЭМ и написание чистого, поддерживаемого кода. Это улучшает читаемость, упрощает доработки, ускоряет загрузку сайта и повышает SEO-оптимизацию.

JavaScript: Основы JavaScript, работа с DOM и создание интерактивных элементов для улучшения UX.

Адаптивная и кросс-браузерная верстка: Создаю сайты, которые корректно работают на любых устройствах и во всех браузерах, даже если адаптивность не предусмотрена в макете. Это важно для удобства пользователей и может напрямую влиять на продажи, так как большинство посещений сейчас происходит с мобильных устройств.

CSS Grid и Flexbox: Эффективное применение CSS Grid и Flexbox для создания гибких и адаптивных макетов.

CSS-препроцессоры: Если требуется использую SCSS и LESS для написания модульного и масштабируемого кода, что упрощает поддержку и ускоряет процесс разработки за счет удобных функций препроцессоров, таких как вложенность.

Инструменты сборки: Использование Gulp и Webpack для автоматизации процессов разработки, оптимизации и минификации кода, что ускоряет загрузку страниц и улучшает производительность сайта.

CSS-фреймворки: Работала с Tailwind CSS для быстрой и эффективной разработки интерфейсов. Этот фреймворк позволяет создавать страницы с минимальными трудозатратами, что ускоряет процесс разработки.

WordPress и PHP: Уверенное создание сайтов на WordPress, разработка тем с нуля, кастомизация шаблонов и настройка функционала с использованием PHP.

Docker: Опыт работы с Docker для создания и управления контейнерами, что упрощает развертывание и тестирование проектов в изолированных средах.

Git и GitLab: Активное использование Git для контроля версий и управления кодом. Опыт работы с GitLab, что позволяет автоматизировать процессы тестирования и деплоя.
